# Duan Yongping Skill

そのまま使える AI スキルツールキット——段永平の投資問答録からビジネスロジックと投資方法を、15 個の単独トリガー可能・組み合わせ呼び出し可能な判断モジュールに分解しました。

これは本の要約ページではありません。本はこれらのスキルの源泉であり、このページの主役ではありません。

## このスキル群が解決する問題

- 企業が投資に値するか判断したいが、どこから見ればいいかわからない
- 「良いビジネスモデル」と「良い製品」の違いが区別できない
- 「正しいことをしている」のか「物事を正しくやっている」のかが分からない
- 自分のストップドゥイングリストを作りたいが、何を除外すべきかわからない
- 集中か分散か、保有か売却かで迷っている

## 何が蒸留されたか

『段永平投資問答録』（ビジネスロジック編 + 投資ロジック編）から 15 個の三重検証済みスキルを抽出し、二つの部分に分けています：

### ビジネスロジック編（7 個のスキル）

- **基盤哲学**：`right-things-first`（正しいことをする — 二元判断）、`stop-doing-list`（ストップドゥイングリスト構築法）
- **投資の大綱**：`right-business-people-price`（投資の三要素 B>P>P）、`business-model-evaluator`（ビジネスモデル評価）、`culture-judger`（企業文化判断）
- **競争戦略**：`dare-to-follow`（敢えて後から追従する）、`profit-above-pursuit`（利益を超えた追求）

### 投資ロジック編（8 個のスキル）

- **投資哲学**：`ownership-mindset`（株を買うことは会社を買うこと）、`stop-doing-list`（除外法による投資規律）
- **分析方法**：`capability-circle`（能力圏の意思決定）、`good-business`（良いビジネスの識別）、`dcf-valuation`（DCF思考によるバリュエーション）
- **意思決定フレームワーク**：`opportunity-cost`（機会コストによる意思決定）、`concentrated-investing`（集中投資）
- **心理管理**：`equanimity-mindset`（平常心で投資する）

## 効果例

### 例 1：ある会社に投資すべきかどうか

**ユーザーの質問**

> 「製品が良く、ユーザー増加が速い会社がありますが、まだ黒字化していません。投資に値しますか？」

**スキルの判断**

- まず `right-things-first` をトリガーし、正しいことをしているのか、それとも物事を正しくやっているだけなのかを確認
- `right-business-people-price` を呼び出し、B>P>P の三要素で分解
- `business-model-evaluator` でビジネスモデルを深く分析：差別化はあるか？堀はあるか？価格決定力はあるか？
- `ownership-mindset` を補足：もしあなたがこの会社の単独所有者だとしたら、どう考えるか？

**回答例**

> ユーザー増加と黒字化を先に見るのではなく、もっと根本的な問いを立てましょう：このビジネスの差別化はどこにありますか？製品は良くても差別化がなく、ユーザーが補助金やブームで来ているだけなら、成長が速いほど将来の崩落も大きくなります。次にビジネスモデルを調べます：何が競合他社のコピーを防いでいるか？価格決定力はどこから来るか？この二つの問いに明確に答えられなければ、製品がいくら良くても良いビジネスではありません。最後に所有者の視点を取り入れ：もしあなたがこの会社を完全に買い取り、将来のキャッシュフローで投資を回収するとしたら、いくら払いますか？答えが「追加資金調達」や「買収されること」に依存しているなら、それは投資対象ではなく、ギャンブルです。

### 例 2：ロスカットして売るべきか

**ユーザーの質問**

> 「買った株が 30% 下落しました。ロスカットすべきですか？」

**スキルの判断**

- `opportunity-cost` をトリガーし、機会コストのフレームワークで買い・売り・保有の基準を統一
- `equanimity-mindset` を呼び出し、感情に駆られて決断していないかを確認
- `capability-circle` を補足し、当初の購入が本当に自分の能力圏内にあったかを確認

**回答例**

> ロスカットの問題は「どれだけ下落したか」ではなく、「もし今手元に現金があったら、この株をまだ買うかどうか」です。買うなら、30% の下落はむしろより良い機会です。買わないなら、当初の購入理由はすでに変化しており、いくら損をしたかとは無関係です。ほとんどのロスカットは感情に駆られています：未実現損失の不快感が「苦痛を終わらせたい」と思わせるのです。しかし、投資判断の唯一の基準は機会コストです：あなたのお金がどこで最高のリターンを得られるか。購入価格を忘れ、現在の価値だけを見てください。

## これらのスキルはどう生成されたか

これらのスキルは **cangjie-skill** の RIA-TV++ パイプラインを使用して生成されました。全行程は六つの段階に分かれています：

1. **全書理解（Adler 分析）**——Mortimer Adler の分析的読書法を用い、全書を構造・解釈・批判・応用の四ステップで分解。`BOOK_OVERVIEW.md` を产出
2. **並列抽出**——5 つの専門抽出器（フレームワーク、原則、事例、反例、用語）を同時に配備し、原文から候補方法論ユニットを抽出
3. **三重検証スクリーニング**——各候補は三つの検査に合格する必要がある：書中に少なくとも 2 箇所の独立した裏付けがある（クロスドメイン）、書中で明示されていない新しい問いに答えられる（予測力）、常識ではない（独自性）
4. **RIA++ 構築**——検証済みの内容を六つの次元で構造化：R（原文引用）/ I（自分の言葉で書き直し）/ A1（書中の事例）/ A2（将来のトリガーシナリオ）/ E（実行可能ステップ）/ B（境界と盲点）
5. **Zettelkasten リンク**——スキル間の依存・対比・組み合わせ関係を見出し、`INDEX.md` と参照グラフを生成
6. **ストレステスト**——各スキルに囮問題を含むテストケースを設計。不合格のスキルは作り直し
