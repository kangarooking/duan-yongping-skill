## 1. 结构 (Structural)

### 类型
方法论 + 实操手册（问答体裁），兼具哲学思辨与案例教学。

### 一句话主旨
买股票就是买公司——以企业所有者视角，在能力圈内识别具有持久竞争优势的好生意，以合理价格买入并长期持有。

### 骨架 (主要论点及其关系)

1. **投资理念的根基**: "买股票就是买公司"是所有投资行为的出发点，由此推导出 Stop Doing List（不做空、不用 margin、不碰不懂的、不走捷径、不盲目扩能力圈、少决策、不短期投机、不碰高负债）
2. **价值投资的基础认知**: 价值投资不等于长期持有，而是以低于内在价值的价格买入；理解价值投资需要理解"市场先生"和"安全边际"
3. **理解公司 = 理解生意**: 投资决策的质量取决于对公司商业模式的理解深度，"看懂一个公司"意味着能判断其未来 5-10 年的盈利能力
4. **财务是验证工具而非起点**: 财报用来验证商业模式判断，而非替代商业理解；ROE/ROA、回购vs分红、资产负债结构是关键验证维度
5. **估值是思维方式而非公式**: DCF 是思考框架（未来现金流的折现），不是 Excel 模型；PE 有严重局限；买卖时机由机会成本决定
6. **方法论层面: 宏观无关、平常心、集中投资**: 不预测宏观经济，保持理性情绪，在能力圈内集中下注
7. **实战案例验证**: 茅台（定价权 + 供需结构）、网易（极低估值 + 游戏业务拐点，100 倍回报）、苹果（生态护城河 + 单品模式 + 平台经济）

**论点之间的关系**: 递进 + 层层深入。从理念根基(1)出发，建立认知框架(2-3)，掌握分析工具(4-5)，形成方法论(6)，最后用案例验证(7)。每一层依赖前一层的结论。

### 作者要解决的核心问题
普通人如何建立一套完整的、自洽的、可执行的投资决策框架——从"想什么"到"怎么想"到"怎么做"。

## 2. 解释 (Interpretive)

### 关键术语 (作者本人的用法)

| 术语 | 作者的定义 | 和常识用法的差异 |
|---|---|---|
| 买股票就是买公司 | 买入股票等同于买入该企业的部分所有权，应以企业所有者视角思考 | 常识将股票视为交易筹码，段永平强调的是所有权思维 |
| 能力圈 | 你真正理解其商业模式和未来盈利能力的公司范围 | 常识以为是"熟悉的行业"，段永平强调的是能判断 5-10 年前景 |
| Stop Doing List | 明确"不做什么"的负面清单，比"做什么"更重要 | 常识偏向正面策略，段永平强调排除法的优先级 |
| 平常心 | 面对市场波动时的理性态度，不被恐惧和贪婪驱动 | 常识理解为"淡定"，段永平指的是基于理解而非情绪的投资决策状态 |
| 未来现金流折现 | 一种思维方式——思考公司未来能产生多少现金，而非计算公式 | 常识以为是 DCF 计算模型，段永平强调它是认知框架 |
| 好生意 | 具有持久竞争优势、能持续产生高资本回报率的商业模式 | 常识以为"赚钱的公司"，段永平强调的是竞争壁垒和长期确定性 |
| 机会成本 | 每个投资决策都应与"最好的替代选项"比较 | 常识以为是会计概念，段永平将其作为所有买卖决策的核心标准 |
| E/E (Enterprise Value / Earnings) | 企业价值倍数，比 PE 更合理的估值指标 | 常识更熟悉 PE，段永平指出 PE 忽略了资本结构差异 |

### 核心命题 (用自己的话)

1. 投资的唯一正确姿势是把自己当成企业所有者，关注企业长期创造价值的能力
2. "不做什么"比"做什么"更重要——Stop Doing List 是投资者的第一道防线
3. 理解一家公司意味着能预判其未来 5-10 年的商业图景，否则就不在能力圈内
4. 估值不是数学题，而是一种商业判断力——核心是理解生意的质量和持续性
5. 最好的投资策略是在能力圈内集中投资，因为分散化是"对无知的保护"
6. 情绪管理（平常心）和认知能力同样重要——恐惧和贪婪是投资最大的敌人
7. 所有投资决策的终极标准是机会成本：这个选择是否优于我目前最好的替代选项

### 论证链
段永平的论证路径是：**信念 → 排除 → 聚焦 → 工具 → 验证**。

1. 先建立核心信念（买股票=买公司），这是公理
2. 由公理推导出 Stop Doing List（如果买公司，就不应该做空、加杠杆等）
3. 排除之后，聚焦于"什么才是好公司"——能力圈 + 商业模式
4. 然后引入工具（财务分析、估值方法），但强调工具服务于判断
5. 最后用三个实战案例（茅台/网易/苹果）验证以上框架的可行性

## 3. 批判 (Critical) ★

### 作者的时代局限
- 本书内容横跨 2006-2018 年左右，中国市场结构（注册制、北交所等）已发生重大变化
- 案例集中于消费（茅台）和科技（苹果/网易），未涉及周期股、金融、公用事业等品类
- 讨论基于美股 + 港股 + 少量 A 股经验，对纯 A 股投资者的适用性需要自行调整
- 量化交易、ETF、指数投资等现代投资工具未被讨论

### 作者的立场盲点
- **成功者偏差**: 网易 100 倍回报是极端案例，段永平自己也说"这辈子可能不会再遇到"，但读者容易将其当作可复制的模板
- **后视镜效应**: 对茅台、苹果的分析是事后的，投资决策时的信息不完备性被低估
- **资金体量差异**: 段永平作为大型投资者的操作逻辑（集中、长期、不择时）对小资金投资者未必最优
- **幸存者偏差**: 只展示了成功的投资案例，失败或犹豫未投的案例讨论较少

### 未被证明的假设
- "好公司长期股价一定会反映价值"——市场非理性的持续时间可能远超预期
- "能力圈可以清晰界定"——现实中能力圈边界是模糊的，"知道自己在能力圈内"本身就需要判断力
- "商业模式分析可以替代宏观判断"——某些行业（银行、地产）与宏观高度相关
- "集中投资优于分散"——这是基于特定投资者能力的前提，对普通投资者分散化可能更安全

### 最强反对意见
1. **指数投资派**: 如果大多数主动投资者无法持续跑赢指数，那段永平倡导的"深入研究+集中投资"对普通人而言是过度自信——巴菲特自己也推荐标普 500 指数基金给大多数人
2. **行为金融学派**: "平常心"的假设违反人类认知偏误的科学研究，实际操作中投资者几乎不可能完全理性，应该用制度设计（如定期定额）而非依赖意志力
3. **量化学派**: 商业模式判断是主观的、不可量化的，而量化策略可以系统性地捕捉市场异常

> **以上批判会直接成为下游 skill 的 Boundary (B) 字段来源**
