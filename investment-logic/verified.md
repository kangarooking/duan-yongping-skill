  passed: true
  novel_question: "股市崩盘，我持有的好公司跌了40%，该不该卖？"
  derived_answer: "如果你是这家公司的唯一股东（非上市公司），今天有人出价比昨天低40%要买你的公司，你会卖吗？如果公司的生意没变，你不会卖。股价下跌不改变公司的未来现金流——'如果只有一只股票而且满仓，真正了解的话下跌就和你无关'。"

V3_exclusivity:
  passed: true
  why_not_common: "'买股票=买公司'这句话本身不算独特，但段永平的行为推论极其反直觉：①不看盘、②不关心短期股价、③把公司抽象为'只有一个股东的非上市公司'来思考。大多数'知道'这句话的人在行为上仍把股票当筹码——段永平说'骨子里真正理解的不超过5个'。"

V2_predictive_power:
  passed: true
  novel_question: "我是医生，理解医药行业，应该投资创新药公司吗？"
  derived_answer: "行业熟悉≠在能力圈内。能力圈的标准是'能判断未来5-10年盈利能力'。理解药物机制不等于能判断商业前景（定价、竞争、审批、医保）。此外，安全边际=理解度——你对创新药商业化的理解有多深，仓位就该多小（如果你只懂医学不懂商业，理解度不够，不应重仓）。"

V3_exclusivity:
  passed: true
  why_not_common: "常识的'能力圈'='熟悉的领域'。段永平重新定义为'能判断未来现金流的范围'。更独特的是'安全边际=理解度'——将格雷厄姆的价格折扣重新定义为认知深度。仓位大小反映理解度而非价格便宜程度，这是对传统投资学的根本性修正。"

  novel_question: "Costco（好市多）是好生意吗？"
  derived_answer: "用好生意框架：①利润和净现金流一直杠杠的？→ 是，会员费模式现金流极稳定；②竞争对手长期难以抢？→ 规模+供应链+会员忠诚度构成壁垒，是；③护城河持久？→ 会员续约率>90%，持续数十年，是。结论：好生意。→ 这与段永平买苹果的逻辑一致（他称苹果是'梦寐以求但难以达到的生意模式'）。"

  why_not_common: "'看商业模式'不是常识——大多数散户从PE/股价/消息开始分析。段永平的分析顺序（商业模式→企业文化→价格）和具体标准（护城河持久性+净现金流+竞争对手能否抢）是巴菲特-段永平学派的独特框架。'定性远比定量重要'更是反华尔街主流。"

## skill-05: DCF思维估值法（未来现金流折现思维方式）

title: DCF思维估值法（含毛估估）
type: framework
merged_from: [f05, f11, p08]
source_chapters: [第1章, 第5章, 第7章]

V1_cross_domain:
  passed: true
  evidence:
    - 第1章: "买公司就是买其未来现金流的折现"作为公理
    - 第5章第1-2节: DCF专题 + Yahoo案例教学 + 毛估估方法
    - 第7章: 苹果（猜5年后利润500亿）、网易（没认真估值但一眼看出便宜）

V2_predictive_power:
  passed: true
  novel_question: "某公司PE=8，看起来很便宜，能买吗？"
  derived_answer: "PE是倒后镜——分子（价格）反映未来预期，分母（利润）是过去数据。PE=8可能意味着：①盈利即将大幅下滑（周期股顶点），看起来便宜实际极贵；②公司确实被低估。用DCF思维：不看PE，而是思考'这个公司未来10年能赚多少钱'。如果能判断未来现金流远大于当前市值，才是真便宜。如果判断不了→看不懂不碰。"

V3_exclusivity:
  passed: true
  why_not_common: "金融教育教DCF作为计算公式，段永平明确否定公式用法。'毛估估'——'300斤的胖子不用秤就知道很胖'——将估值从数学题变成商业判断力测试。'想用公式的请到学校去用'是反学术的独到表述。"

id: skill-06
title: 机会成本决策框架（含持有=买入）
type: framework
merged_from: [f07, f13, p09, p13]
source_chapters: [第1章, 第5章, 第7章]

V1_cross_domain:
  passed: true
  evidence:
    - 第1章: "做对的事情"纠错逻辑
    - 第5章第5-6节: "持有=买入"论证 + 机会成本专题
    - 第7章: 茅台换仓案例、网易卖出时机

V2_predictive_power:
  passed: true
  novel_question: "我50元买的股票现在100元，翻倍了，该不该止盈？"
  derived_answer: "成本价50元是沉没成本，与决策无关。问题变成：如果你今天有100元现金，你愿意按当前价格买入这只股票吗？如果愿意→持有（=买入）。如果不愿意→卖出，因为持有=用当前价格买入。止盈逻辑（'已经赚了'）是段永平明确反对的唯一卖出理由。应该比较：持有vs 你最好的替代投资机会。"

V3_exclusivity:
  passed: true
  why_not_common: "'持有=买入'是段永平最独特的表述之一——它把'被动持有'变成了'主动决策'，把投资从一次性决策变成了每天的选择。'卖股票和成本无关'也反直觉（大多数人的卖出决策深受买入价格影响）。'没有卖就等于买了'是刷新认知的视角。"

V2_predictive_power:
  passed: true
  novel_question: "我的股票涨了200%，开始焦虑要不要卖，这是正常的吗？"
  derived_answer: "焦虑说明你可能在投机而非投资。如果你真正理解这家公司（能力圈内），你应该知道它值多少钱。涨200%后如果仍然低于内在价值→持有，因为还有上涨空间；如果已远超内在价值→卖出。焦虑来自'不知道值多少钱'——这是对理解度不足的信号。解决方法不是压制情绪，而是加深理解。"

V3_exclusivity:
  passed: true
  why_not_common: "常识的'平常心'='不激动/淡定/佛系'。段永平重新定义为'回到事物本源'——是主动的认知行为（找到本质）而非被动的情绪压制。路径也很具体：①用闲钱（物质前提）②深入理解（认知前提）③不比较（社交前提）。'在错的道路上没办法有平常心'——平常心是正确方法论的副产品，不是可以独立追求的状态。"

id: skill-08
title: 集中投资框架

  passed: true
  evidence:
    - 第2章: "绝对的集中。分散是理财不是投资"
    - 第3章第4节: 集中投资+能力圈逻辑
    - 第6章: "一辈子做20个投资决策就够了"

V2_predictive_power:
  passed: true
  novel_question: "我有100万，应该买10只股票各10万来分散风险吗？"
  derived_answer: "分散化是'对无知的保护'——如果你需要买10只股票来分散风险，说明你对每只都不够了解。如果你真正理解其中2-3只（在能力圈内），集中投那2-3只。仓位大小应反映理解度（安全边际=理解度）。100万投3只真正懂的 > 100万投10只似懂非懂的。段永平自己就投了几家公司而已。"

V3_exclusivity:
  passed: true
  why_not_common: "常识是'不要把鸡蛋放在一个篮子里'。段永平明确反转——集中投资优于分散。'分散是理财不是投资'、'投的企业越多赚的越少'都是反常识的强断言。这与能力圈逻辑闭环：正是因为真正懂的公司很少，所以必须集中；如果需要分散，说明你的能力圈定义有问题。"
