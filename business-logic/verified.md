## 通过验证的单元（7个 → 进入阶段 2）

### 1. Stop Doing List 不为清单构建法

**来源候选**: f03, p31-p48, p56, ce06-ce12
- V1_cross_domain:
  passed: true
  evidence:
    - 第1节: "好公司都一定是有一个长长的Stop doing list"
    - 第3节: "做对的事情实际上是通过不做不对的事情来实现的"
    - 第7节: 整节20+条具体Stop Doing条目，每条都有故事和逻辑
    - 第6节: "不多元化"作为Stop Doing的延伸
- V2_predictive_power:
  passed: true
  novel_question: "一个快速增长的中型SaaS公司应该如何控制扩张节奏？"
  derived_answer: "先列Stop Doing List：不借钱扩张、不做自己不理解的细分、不追求市场份额排名、不打价格战。用'足够的最小发展速度'替代最大速度。"
- V3_exclusivity:
  passed: true
  why_not_common: "绝大多数商业建议聚焦'做什么'（增长策略、机会清单），段永平反其道而行之——核心竞争力是不做什么。这不是消极的'不做'，而是主动的战略排除。Stop Doing List有具体的构建方法（两类：普世禁忌+使命绑定）、验证方式（发现错了立刻停）和衡量标准（坚持10-20年后看差距）。"

### 2. 商业模式评估：差异化→护城河→定价能力

**来源候选**: f06, f21, p05-p09, p40, p58, ce01-ce05
- V1_cross_domain:
  passed: true
  evidence:
    - 第2节: 差异化是好商业模式的前提（航空vs白酒案例）
    - 第4节: 产品差异化分析（同质化→价格战→不赚钱）
    - 第2节案例点评: 茅台定价能力、苹果生态系统、航空公司零差异化
- V2_predictive_power:
  passed: true
  novel_question: "社区团购是一个好生意吗？"
  derived_answer: "看差异化：消费者对买菜的核心诉求是新鲜+便宜+方便，差异化空间极小→护城河浅→难以建立定价能力→长期难赚大钱。短期靠补贴获客不是可持续差异化。"
- V3_exclusivity:
  passed: true
  why_not_common: "常见的商业分析只讲'竞争优势'这个笼统概念。段永平的三段链（差异化→护城河→定价能力）是可操作的递推逻辑，加上'能涨价=有护城河'这一简单验证法，构成了一套快速判断工具，这是他的独特贡献。"

### 3. "做对的事情"二元决策框架

**来源候选**: f01, f15, f17, f25, f27, p03, p13, p14, p17
- V1_cross_domain:
  passed: true
  evidence:
    - 第1节: "好公司是知道要做对的事情的公司"
    - 第3节: "本分=做对的事情+把事情做对"，发现错了立刻停
    - 第7节: Stop Doing List的哲学根基就是"通过不做不对的事来做对的事"
    - 前言: "买股票就是买公司"本身就是"做对的事情"的具体应用
- V2_predictive_power:
  passed: true
  novel_question: "一个程序员纠结要不要接一个高薪但不认同其商业模式的工作，怎么决策？"
  derived_answer: "先问'这是对的事情吗'（方向判断）——如果不认同其商业模式，就是在做不对的事情；再问'我能不能把它做好'（执行判断）——即使做好也改变不了方向错误。做对的事情优先于把事情做对，方向错了执行越好结果越差。"
- V3_exclusivity:
  passed: true
  why_not_common: "常识说'努力把事情做好'，段永平说'方向比努力重要'。将决策明确分为'方向判断'（道/对错二元）和'执行效率'（术/高低连续）两个维度，并强调顺序不可颠倒——这不是常见的思维框架。"

### 4. 企业文化判断框架

**来源候选**: f04, f23, p04, p10-p21, p52, p54, p55, p57, ce07, ce08, ce13-ce18
- V1_cross_domain:
  passed: true
  evidence:
    - 第1节: "没有强大的文化，就不会有伟大的公司"（5-10年CEO,10-50年董事会,50年+文化）
    - 第3节: 本分/平常心/诚信/消费者导向的详细阐述
    - 第7节: Stop Doing List是企业文化的外在表现（"这就是所谓的企业文化"）
    - 第3节案例点评: 松下文化衰退、索尼质量下降、阿里巴巴文化
- V2_predictive_power:
  passed: true
  novel_question: "如何从外部判断一家未上市公司的企业文化好坏？"
  derived_answer: "1）看创始人：信不信自己的价值观（看他历史中说的和做的）；2）看负面：做了什么'不对'的事比做了什么'对'的事更容易判断；3）看人才：高管是从内部提拔还是空降（合适性>合格性）；4）看细节：员工离职时年终奖是否照发、供应商货款是否按时付。"
- V3_exclusivity:
  passed: true
  why_not_common: "'本分'不是字典里的安分守己，而是段永平独创的概念体系：做对的事情+发现错了立刻停+求责于己+不赚人便宜。'平常心'也不是佛系的消极心态，而是理性的思考习惯——排除干扰回到本质。这些定义和用法是段永平体系独有的。"

### 5. Right Business / Right People / Right Price 投资三要素

**来源候选**: f02, f13, f24, p01, p05, p06, p53
- V1_cross_domain:
  passed: true
  evidence:
    - 前言: "买股票就是买公司"（Right Business的具体化）
    - 第2节: "right business, right people, right price"直接引用，贯穿商业模式讨论
    - 第3节: "right people"=企业文化
    - 第7节: Stop Doing List本身就是对Right Business的筛选实践
- V2_predictive_power:
  passed: true
  novel_question: "一家商业模式很好但价格很贵的公司（PE=80），值得投资吗？"
  derived_answer: "Right Business（✓）+ Right People（待验证）+ Wrong Price（✗）→ 价格不重要因为时间会降低其重要性，只要前两个对、价格不是极端贵就可以。但前提是前两个确实对。如果PE=80是因为市场疯狂而非基本面，时间会站在你这边。"
- V3_exclusivity:
  passed: true
  why_not_common: "一般投资建议会说'看基本面'，但段永平给出了明确的优先级排序（business>people>price）和理由（时间降低价格重要性、好生意让庸才都不怕）。这不是泛泛的'价值投资'，而是一套有操作优先级的过滤框架。"

### 6. "敢为天下后"后发制人策略

**来源候选**: f11, p24, ce15, c10, c14
- V1_cross_domain:
  passed: true
  evidence:
    - 第4节: "没有差异化的'敢为天下后'是无法生存的"
    - 第6节: 步步高从学习机→VCD→手机的每次后发进入都验证了这一策略
    - 第7节: "不弯道超车"与"敢为天下后"一脉相承
- V2_predictive_power:
  passed: true
  novel_question: "一个中国团队想做AI大模型创业，应该做基础模型还是应用层？"
  derived_answer: "基础模型是先行者游戏（烧钱培育市场），应用层是'敢为天下后'——等基础模型成熟后找到差异化切入点。前提是你能提供差异化（如行业know-how、特定场景优化）。没有差异化就后发进入等于送死。"
- V3_exclusivity:
  passed: true
  why_not_common: "商业世界的默认建议是'抢占先机'（first mover advantage）。段永平反其道而行：小企业培育市场代价太大，等市场成型后集中兵力切入更高效。关键是'后中争先'——不是消极等待，而是看清后精准出击。这是反直觉的。"

### 7. "利润之上的追求"企业诊断

**来源候选**: f12, p02, p11, p12, p22, ce09, ce10
- V1_cross_domain:
  passed: true
  evidence:
    - 第1节: 乔布斯/苹果案例——"动力来自产品而不是利润"
    - 第3节: "企业文化指的就是利润之上的追求"
    - 第3节案例: 索尼从产品导向变利润导向→质量下降
    - 第7节: "不赚快钱"是利润之上追求的具体实践
- V2_predictive_power:
  passed: true
  novel_question: "一家公司每年都优化成本、压榨供应商来提高利润率，但产品质量在下降。这家公司长期会怎样？"
  derived_answer: "碰到问题时先问'赚不赚钱'而非'这对不对'，说明缺乏利润之上的追求→短期利润好看但品牌和消费者信任在流失→20年后差别会很大。"
- V3_exclusivity:
  passed: true
  why_not_common: "一般说法是'企业要有社会责任'，这太空泛。段永平给出了具体的诊断方法：看公司面对问题时先问'赚不赚钱'还是先问'这对不对'。这个具体的诊断信号是独特的。而且他强调这不是充分条件（有追求不一定是好企业），但时间越长与一般企业差距越大——这个'时间放大器'的洞见是独特的。"
